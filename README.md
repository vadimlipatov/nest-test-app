# Тестовое задание

## Description

API Написано на NestJS, c использованием axios, custom pipes

## Downloading

```bash
# clone or download from github
$ git clone https://github.com/vadimlipatov/test-app.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Request Example 

```bash
# get all posts
$ http http://localhost:3002/api/posts

# get one post
$ http http://localhost:3002/api/posts/1
```
